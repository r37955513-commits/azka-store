# 🛍️ أذكى متجر — متجر الخدمات الرقمية

متجر إلكتروني متكامل للخدمات الرقمية (شحن الألعاب، بطاقات الهدايا، اشتراكات السوشال ميديا)
مبني بـ **React Router 7** + **Hono** + **PostgreSQL (Neon)** + **Tailwind CSS** + **Lucide Icons**.
واجهة عربية بالكامل تدعم الكتابة من اليمين لليسار (RTL) مع تنبيهات Toaster.

---

## ✨ الميزات

- **واجهة العرض** — صفحة رئيسية تعرض المنتجات مقسّمة حسب الفئات (ألعاب، تطبيقات، بطاقات).
- **نظام المحفظة** — محفظة رقمية لكل مستخدم (حسب رقم الهاتف)، مع ميزة "طلب شحن رصيد" ورفع صورة إشعار التحويل.
- **نظام الطلبات** — الخصم من المحفظة أو الطلب اليدوي، مع حالات (قيد الانتظار / مكتمل / ملغي) واسترجاع تلقائي للرصيد عند الإلغاء.
- **نظام الكوبونات** — خصم نسبة مئوية أو مبلغ ثابت مع التحقق من الصلاحية، الحد الأدنى للطلب، وعدد الاستخدامات.
- **لوحة تحكم الأدمن** (محمية بكلمة مرور):
  - إدارة الطلبات وتغيير حالتها.
  - مراجعة طلبات شحن المحفظة وعرض صور الإشعارات والموافقة لإضافة الرصيد تلقائياً.
  - إدارة الكوبونات والإحصائيات (إجمالي المبيعات، عدد الطلبات).
- **إشعارات WhatsApp** — إشعار تلقائي للأدمن عبر WhatsApp Cloud API عند كل طلب جديد أو طلب شحن.

## 🚀 التشغيل

### 1. المتطلبات
- Node.js 20+
- قاعدة بيانات PostgreSQL على [Neon](https://neon.tech)

### 2. التثبيت
```bash
npm install
cp .env.example .env
```

### 3. إعداد قاعدة البيانات
```bash
npm run db:setup
```

### 4. التطوير
```bash
npm run dev
```

### 5. الإنتاج
```bash
npm run build
npm start
```

## 🔑 متغيرات البيئة (.env)

| المتغير | الوصف |
|---|---|
| `DATABASE_URL` | رابط اتصال Neon PostgreSQL |
| `ADMIN_PASSWORD` | كلمة مرور لوحة التحكم |
| `SESSION_SECRET` | سرّ توقيع جلسات الأدمن |
| `WHATSAPP_TOKEN` | توكن WhatsApp Cloud API |
| `WHATSAPP_PHONE_ID` | معرّف رقم WhatsApp المُرسِل |
| `ADMIN_WHATSAPP_NUMBER` | رقم الأدمن لاستقبال الإشعارات |
| `PORT` | منفذ خادم Hono في الإنتاج |

## 🗄️ هيكل قاعدة البيانات

| الجدول | الوصف |
|---|---|
| `categories` / `products` | الفئات والمنتجات |
| `users` | المستخدمون (حسب رقم الهاتف) |
| `wallets` / `wallet_transactions` | المحافظ وسجل الحركات |
| `orders` | الطلبات وحالاتها |
| `topups` | طلبات شحن المحفظة وصور الإشعارات |
| `coupons` | الكوبونات |

> 🔐 الوصول للوحة التحكم: `/admin/login`
